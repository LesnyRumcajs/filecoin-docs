Typically, to buy {{< tooltip "FIL" >}} from an {{< tooltip "exchange" >}} exchange, you would:

1. Create an account with an exchange.
1. Complete the exchanges verification and Know-Your-Customer (KYC) process, if any.
1. Add funds to your account.
1. Exchange your funds for FIL or another cryptocurrency.

## Exchanges that support FIL

Not all exchanges support FIL. For current information on exchanges that support FIL, see:

- [coingecko.com](https://www.coingecko.com/)
- [coinmarketcap.com](https://coinmarketcap.com/currencies/filecoin/markets/)

## Things to note

Note the following when choosing an exchange:

- With exchanges that do support FIL, support for [FIL address types]({{< relref "/basics/the-blockchain/addresses" >}}) varies between exchanges.
- Potential downsides of using an exchange include hacks and regulatory uncertainty.
- Features, usage fees, security protocols, and supported cryptocurrencies vary between exchanges.

As such, users should do their own research before using any particular exchange.
